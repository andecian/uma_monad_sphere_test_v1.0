# uma_monad_sphere_test_v1.0
UMA Monad Sphere (Test)

## Overview
This is a dynamic 3D sphere simulation tool based on the **UMA Monad Theory** proposed by Hirofumi Miyauchi. By manipulating the sliders for the additive circle (Cx) and multiplicative circle (Cy), users can visually and mathematically observe how the existence quantity (Cz) fluctuates.

**Note on Current Status:**
The current behavior of this simulation may be experimental or slightly irregular. It is being released at this stage primarily to facilitate the sharing of the visual concept and the underlying intuition behind the theory.

## Features
- **Standalone Architecture**: Developed using pure HTML, Vanilla JS, and CSS with no external dependencies.
- **Real-time Mathematics**: Dynamically calculates coordinates on the unit sphere ($Cx^2 + Cy^2 + Cz^2 = 1$) and provides real-time verification.
- **Phase Identification**: Automatically describes the current state (e.g., UMA Monad, Pole of Presence, Equator) based on the variables.

## Usage
1. Download the `uma_monad_sphere_text_vx.x.html` file.
2. Open it in a modern web browser (e.g., Chrome, Safari, Firefox) to run the explorer locally.
